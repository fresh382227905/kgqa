# 1.4亿通用知识图谱问答平台开发教程

一步步教你搭建1.4亿通用知识图谱问答平台，共4572万实体，1.41亿实体关系，详细目录如下：
* 数据集下载及预处理
* 导入到neo4j图数据库
* 图谱问答预测流程介绍
* 生成训练样本
* 训练模型
* 相似度匹配
* Flask后端发布
* Vue前端开发
* 特别鸣谢

## 数据集下载及预处理
1.4亿三元组数据集来自ownthink开源的中文知识图谱，点击[这里](https://github.com/ownthink/KnowledgeGraphData)下载，数据是以（实体、属性、值），（实体、关系、实体）混合的形式组织，数据格式采用csv格式, 打开详情如下：
```
$ head ownthink_v2.csv
```
